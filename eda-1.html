EDA I - Methodology

Exploring new data in a systematic approach

Data Overview

Initial dataset exploration

Explore shape, features, target variable and missing values

Univariate Feature Analysis + Target Correlation

Comprehensive analysis for first model run

  1. Understand data distribution with basic statistics
    • mean, std dev, percentile etc.
    • frequency distribution, cardinality
  2. Visualize outliers and distribution shape
  3. Assess correlation, statistical significance and effect size against target variable
    • Mann-whitney U test, Cohen's D
    • Crosstab, Chi-square, Cramer's V
  4. Experiment on different ways to preprocess data

Bivariate/Multivariate Analysis + Target Correlation

Data deep dive to improve model performance

  1. Detect multicollinearity (correlation matrix)
  2. Gather insights for feature engineering
    • Life stage (age + occupation + marital status + education)
    • Channel/communication effectiveness (contact method + previous contact days, campaign calls)
    • Life stage + channel communication effectiveness
    • Financial status (housing loan, personal loan, credit default)
