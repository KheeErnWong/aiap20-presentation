🆔 Client ID Deep Dive Analysis
×
📊 Initial Findings
  • • Client ID showed strong correlation (0.746) with subscription rates
  • • Clear upward trend: from 2.8% (D1) to 45.9% (D10) subscription rates across deciles
📈 Client ID Quantiles vs Subscription Rates
🔍 Evaluation
  • ❌ False Pattern: High customer IDs correlated with success, but the ID itself doesn't cause higher subscription rates, but likely reflects campaign optimization over time. Feature fails as soon as new customers get random/sequential IDs
⚖️ Decision
EXCLUDED Client ID features to prevent data leakage and ensure model generalizability.
📈 Recommended Data Collection Improvements
Instead of relying on Client ID as a proxy for temporal patterns, future data collection should capture explicit temporal features such as:
  • • Campaign start/end dates and duration
  • • Seasonal indicators (holiday periods, bonus season, end-of-quarter timing)
  • • Campaign batch/phase identifiers
  • • Economic context variables (market conditions, interest rates)
  • • Campaign maturity metrics (days since campaign launch)
